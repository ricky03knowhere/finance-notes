
# UI Design Guideline

Gunakan desain yang clean dan premium.

Inspirasi:

* Apple
* Linear
* Notion
* Stripe Dashboard
* Arc Browser

Style:

* Glassmorphism ringan
* Rounded corner 18–24 px
* Shadow halus
* Banyak white space
* Animasi smooth
* Responsive
* Minimalis
* Modern
